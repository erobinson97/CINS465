Steps to making the chess board:

  1. Make a table: table with border="5" and id="chess".
  2. Within the table create 8 rows using <tr>
  3. Within each <tr> create 8 columns using <td>.
  4. Give each <td> an alternating class of either "black" or "white".
  5. Each <td> is a square and using the classes we can color each square purple and yellow.
  6. Within each <td> there is a unicode charater for each chess piece.
    • White King: ♔
    • Black King: ♚
    • White Queen: ♕
    • Black Queen: ♛
    • White Rook: ♖
    • Black Rook: ♜
    • White Bishop: ♗
    • Black Bishop: ♝
    • White Knight: ♘
    • Black Knight: ♞
    • White Pawn: ♙
    • Black Pawn: ♟
  7. Use CSS to give our table with id="chess" an entierly black border color:

    border-color: #000000;

  8. Give the table with id="chess" the following to center it on the page.

    margin-left: auto;

    margin-right: auto;
